# Unit Converter Tool

A simple, efficient Python command-line interface (CLI) tool designed to handle common unit conversions.

## Features
- **Distance Conversion**: Easily convert between Kilometers and Miles.
- **Temperature Conversion**: Perform quick conversions between Celsius and Fahrenheit.
- **Modular Design**: Clean, function-based architecture for easy maintenance and expansion.

## How to Run
1. Ensure you have Python installed on your system.
2. Clone this repository to your local machine.
3. Open your terminal or command prompt.
4. Navigate to the project folder and run:
   ```bash
   python unit_converter.py
